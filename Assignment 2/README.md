## Library Specs and Requirements
- Java Version 24.0.1, Language Level 21 is required.
- Library Version 1.0

## Testing The Library (Test Application)
- Open a CLI, and navigate to the root directory of this assignment (i.e., where src lives).
- Run `javac ./src/lib/*.java -d ./bin`.
- Run `jar cfe LibraryTest.jar lib.LibraryTest -C bin .`.
- Run `java -jar ./LibraryTest.jar` to run the test application.
